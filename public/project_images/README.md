# Project Images Folder

This folder contains all project images for the portfolio gallery.

## How to Add a New Project Image

1. **Add your image file here** (e.g., `my-project.jpg`)
2. **Update `components/Projects/projectsData.ts`** with the image path:
   ```typescript
   imagePath: "/project_images/my-project.jpg",
   ```

## Image Guidelines

- **Recommended size**: 800x600px or 16:9 aspect ratio
- **Formats**: `.jpg`, `.jpeg`, `.png`, `.webp`
- **Naming**: Use lowercase with hyphens (e.g., `ecommerce-platform.jpg`)
- **Quality**: Use high-quality images for best visual impact

## Example

If you add an image named `portfolio-website.jpg`, your path would be:
```typescript
imagePath: "/project_images/portfolio-website.jpg",
```
